# Glossary

## Project

포트폴리오에 표시되는 프로젝트 단위이다.

## ProjectCard

프로젝트 목록 또는 홈 화면에서 사용하는 요약 데이터이다.

관리 위치:

- `src/data/projects.ts`

## ProjectDetail

프로젝트 상세 페이지에서 사용하는 상세 데이터이다.

관리 위치:

- `src/data/projectDetails.ts`

## Technical Note

개발 과정에서 겪은 문제, 분석, 해결, 결과를 기록한 기술 문서이다.

관리 위치:

- `src/data/technicalNotes.ts`
- `src/data/noteDetails.ts`

## slug

라우트와 데이터 연결에 사용하는 고유 문자열이다.
