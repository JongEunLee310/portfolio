# ADR-004: 다크 히어로와 라이트 콘텐츠 카드 디자인 시스템

## 상태

Accepted

## 결정

공통 디자인은 다크 히어로, 라이트 콘텐츠 카드, 블루 포인트 컬러를 기준으로 구성한다.

## 규칙

- Hero와 Footer는 다크 네이비 계열을 사용한다.
- 본문 목록과 카드 영역은 라이트 배경을 사용한다.
- 파란색은 CTA와 핵심 강조에 제한적으로 사용한다.
- 반복 스타일은 `src/styles/classNames.ts` 또는 공통 컴포넌트로 분리한다.

## 예시 이미지 기반 페이지 규칙

사용자가 제공한 로컬 참고 이미지는 Git에 포함하지 않는다. 이미지는 `/Users/sleepyowl/Downloads`에만 두고, 구현에는 아래 관찰된 디자인 규칙만 반영한다.

### 공통

- Header는 다크 배경 위 고정형에 가깝게 보이게 하고, 활성 메뉴는 파란색 텍스트와 하단 라인으로 구분한다.
- Hero는 왼쪽 메시지, 오른쪽 3D 아키텍처 비주얼의 2열 구성을 기본으로 한다.
- 다크 영역은 배경 패턴과 블루 글로우를 사용하되, 본문 가독성을 해치지 않게 절제한다.
- 라이트 본문은 큰 흰색 카드와 얇은 테두리, 부드러운 그림자를 사용한다.
- Footer는 다크 배경, 연락 채널, 이력서 다운로드 CTA를 한 줄 그리드로 배치한다.

### Home

- 첫 화면은 강한 다크 Hero와 2개 CTA를 둔다.
- Featured Projects는 큰 가로 카드 2개를 사용하고, 썸네일과 기술 태그를 함께 보여준다.
- Other Projects는 작고 넓은 카드로 요약 정보를 빠르게 훑게 한다.
- Technical Notes Preview는 다크 섹션의 작은 카드 리스트로 리듬을 바꾼다.
- Tech Stack은 라이트 배경의 미니 카드 그리드로 마무리한다.

### Projects

- Hero 아래에 카테고리 pill 필터를 둔다.
- 목록 화면은 왼쪽 필터 사이드바와 오른쪽 3열 프로젝트 카드 그리드를 기본으로 한다.
- 카드에는 썸네일, 카테고리 배지, 제목, 요약, 태그, 상세/GitHub 버튼을 모두 포함한다.
- 정렬 버튼, 그리드/리스트 토글, 페이지네이션 또는 더 보기 버튼을 위한 자리를 남긴다.

### Project Detail

- Hero에는 프로젝트명, 한 줄 설명, 기간/역할/팀 규모, 핵심 성과 배지, 대표 대시보드 이미지를 배치한다.
- 상세 본문은 프로젝트 개요, 문제 정의, 해결 방향, 아키텍처, 핵심 기능, 스크린샷, 기술 스택, 기여, 성과, 트러블슈팅, 회고 순서를 따른다.
- 아키텍처는 넓은 다이어그램 카드로 처리하고, 성과 지표는 다크 배경의 숫자 카드로 강조한다.

### Technical Notes

- 목록 페이지는 Hero 다음에 카테고리 pill 필터를 둔다.
- 추천 글은 큰 가로 카드로 강조하고, 일반 글은 3열 카드 그리드를 사용한다.
- 카드에는 카테고리 배지, 썸네일, 제목, 요약, 날짜, 읽는 시간을 포함한다.

### Technical Note Detail

- Hero에는 글 제목, 카테고리, 날짜, 읽는 시간, 작성자, 태그를 둔다.
- 데스크톱 본문은 왼쪽 sticky TOC와 오른쪽 글 본문 2열 구성을 사용한다.
- 코드블록은 다크 배경으로, 성능 비교는 숫자 중심의 카드로 보여준다.
- 글 하단에는 관련 글 CTA를 둔다.

### About

- Hero는 개발자 정체성과 프로젝트/연락 CTA를 제공한다.
- 본문 첫 섹션은 프로필 카드와 소개 카드를 조합한다.
- 강점, 경력/교육, 기술 스택, 일하는 방식, 지표 기반 성장, CTA 순서로 구성한다.
- 프로필 이미지는 실제 사진 또는 일러스트를 사용하되, 임시 아바타는 나중에 교체한다.

### Contact

- Hero는 협업 제안 메시지와 연락 관련 3개 가치 배지를 포함한다.
- 본문은 문의 폼 카드와 연락처 정보 카드를 2열로 배치한다.
- 연락처 정보에는 Email, GitHub, LinkedIn, GitLab, Resume 다운로드를 제공한다.
- FAQ는 아코디언 형태를 기준으로 하고, 하단 CTA는 다크 배너로 구성한다.
