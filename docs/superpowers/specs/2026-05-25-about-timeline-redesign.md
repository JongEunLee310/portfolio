# AboutTimeline 중앙 센터라인 재설계

## 목표

경력·교육 타임라인을 가운데 수직선 기준으로 양쪽에 항목을 배치하는 레이아웃으로 전환한다.

## 레이아웃 규칙

- **왼쪽**: `type !== "project"` 항목 (career, education, bootcamp)
- **오른쪽**: `type === "project"` 항목
- **중앙 수직선**: 골드 색상(`--color-accent`), 상단~하단 연속
- **연도 마커**: 연도가 바뀌는 지점에 골드 pill 배지로 표시
- **도트**: 각 항목마다 중앙 선 위에 골드 dot

## 행(Row) 구성

1. 타임라인 데이터를 시작 연도 기준으로 그룹화 (최신순)
2. 각 연도 그룹 내에서 왼쪽 목록과 오른쪽 목록을 분리
3. 두 목록을 zip 방식으로 묶어 행 생성 — 한쪽이 없으면 빈 셀
4. 연도 그룹 앞에 연도 마커 삽입

예시 (2025년):
- 왼쪽: [Kernel360, 가사도우미]
- 오른쪽: [ChatGPT 웹]
- 행1: Kernel360 | • | ChatGPT 웹
- 행2: 가사도우미 | • | (빈 셀)

## 연도 추출 규칙

`period` 필드의 첫 4자리 숫자를 시작 연도로 사용.

## 반응형

- `lg` 이상: 중앙 센터라인 양쪽 배치
- `lg` 미만: 단일 컬럼 (기존 왼쪽 정렬 타임라인, 모든 항목 포함)
